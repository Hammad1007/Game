Pacman game Directions:

*The main purpose of this document is to 
 provide a brief help for those who want  to make this game

Functions to be used in the game:

-> gotoxy function: This function is used for    setting the position of the cursor on the    screen and is used to move the cursor     on the screen.

-> food function: This function prints the      food or maybe the source of points for    the player. It considers the locations on    the map by dealing with the rows and      columns and then printing the dot on       the screen in the map.

-> display map: This function is used to       display the map on the screen.

-> system("cls") is used to clear the       screen

-> srand time is used to genrate the       movement of the enemy randomly
